# JS100

## 1일차! 1-3문제
- `.pop()` : 배열의 맨 끝에 값을 제거
- `.shift()` : 배열의 맨 앞에 값을 제거
- `.push()` : 배열의 맨 끝에 값을 추가
- `.unshift()` : 배열의 맨 앞에 값을 추가한다.
- `.slice(start[, end])` : 배열에 영항을 미치지 않음! 종료인덱스 포함 X
- `.splice(start[, deleteCount[,item1[,item2[, ...]]]])` : 삭제 및 추가할 때! 배열에 영향을 미침
- `typeof` : 자료형 확인
